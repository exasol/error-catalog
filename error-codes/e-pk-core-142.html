Exasol Error Catalog

E-PK-CORE-142

Message

Error starting the 'go-licenses' binary in working dir {{working dir}}.

Mitigations

  • Verify that 'go-licenses' is installed.
  • Install it by running 'go install github.com/google/go-licenses@latest'.
  • If it is already installed, re-install it by running the same command.
  • This might be necessary after installing a new Go version.

    • Parameters

    working dirworking directory where go-licenses was executed
